Check your app interactively
----------------------------

.. _hoppscotch: https://hoppscotch.io/

To test your app, you can use a service like postman (commerical) or
`hoppscotch`_ (open source, freemium)

.. image:: _static/hoppscotch.png

These tools make it easy to test APIs and also convert these tests to
code:

.. image:: _static/hoppscotch-python.png

Now you can use the same code for test your API locally and on AWS.
Just replace the URL with the public end point generatd by AWS lightsail.

.. image:: _static/hoppscotch-live-aws.png
